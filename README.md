Svelte knowledge

- Svelte是编译器，而不是像 React 或 Vue 这样的依赖项
- Svelte 似乎需要更少的代码，用 React 开发同样的功能代码量大约会多 40％ （来源：Rich Harris）
- Svelte 没有 virtual DOM，它会被编译成最小的 “vanilla” JavaScript，并且看起来比其他库性能更好
